## Download Instructions

### Option 1: Manual Download (Recommended)
1. Visit each official brand page
2. Download PNG or SVG (transparent background)
3. Save to `/public/images/logos/tech-partners/`
4. Name format: `stripe.png`, `openai.png`, etc.

### Option 2: Use Brand Asset APIs
Some services provide direct logo URLs:
- Clearbit Logo API: `https://logo.clearbit.com/{domain}`
- Example: `https://logo.clearbit.com/stripe.com`

### Option 3: Create Sprite Sheet
If you prefer to create a single PNG with all logos:
- Layout: 3 columns x 3 rows
- Size: Each logo ~200x80px
- Spacing: 40px between logos
- Total: ~800x400px
- Save as: `tech-partners-sprite.png`

## Sizing Guidelines
- **Width:** 80-120px (desktop), 60-80px (mobile)
- **Height:** Auto (maintain aspect ratio)
- **Format:** PNG or SVG preferred
- **Background:** Transparent
- **Color:** Black or grayscale (for dark mode compatibility)

## Dark Mode Versions
For logos that don't work on dark backgrounds:
- Stripe: Use white version
- OpenAI: Use white version
- Plaid: Use white version
- Others: Invert or use white versions

## Fallback Strategy
If logo doesn't load:
- Display text name instead
- Use brand color for text
- Maintain spacing/layout
