## Why maintenance discipline compounds

Reactive maintenance costs more and damages guest experience. A scheduled approach protects both utilization and brand reputation.

## Build a three-trigger schedule

For each vehicle, define:

- Mileage trigger (for wear-based tasks)
- Time trigger (for fluids and inspections)
- Condition trigger (for observed issues)

Use the earliest trigger as the action point.

## Maintenance priority tiers

| Tier | Description | Action window |
|---|---|---|
| P1 | Safety-critical issue | Immediate |
| P2 | Performance degradation risk | 1-3 days |
| P3 | Preventive routine | Planned within cycle |

## Weekly maintenance review

Track:

- Tasks due this week
- Tasks completed on time
- Tasks missed
- Total downtime from maintenance

This makes reliability measurable and improvable.

## Final takeaway

Maintenance is not overhead. It is a revenue-protection system that keeps vehicles earning.
